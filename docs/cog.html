Namespace: cog

cog

CogJs exposes itself as a global namespace and as a named AMD module called 'cog'. This namespace contains all classes and core utility functions.

<static> defaults(deep, target, object1, objectN) → {object}

Merges the contents of two or more objects into the first object. Target properties will not be overwritten.

Based on jquery extend, but modified to keep existing props instead.

Note: Source arraies will only be copied if the target's property is undefined.

Parameters:
Name Type Argument Description
deep boolean <optional>

Optional, when true, properties will be deep copied.
target object

An object that will receive properties if additional objects are passed in.
object1 object <optional>

An object containing properties to be merged into the target.
objectN object <optional>

Additional objects containing properties to be merged into the target.

<static> extend(deep, target, object1, objectN) → {object}

Merge the contents of two or more objects into the first object. Target properties will be overwritten.

Based on jQuery.extend()

Essentially the same as jQuery.extend, except extends the cog namespaces instead of the jquery namespace if only one argument is passed in.

Parameters:
Name Type Argument Description
deep boolean <optional>

If true, the merge becomes recursive (a.k.a. deep copy).
target object

An object that will receive the properties if additional objects are passed in. Alternatively, target will extend the cog namespace if it is the sole argument.
object1 object <optional>

An object containing properties to be merged into the target.
objectN object <optional>
<repeatable>

Additional objects containing properties to be merged into the target. Will overwrite properties provided by previous objects.
Source:
  • core.js
See:
  • http://api.jquery.com/jQuery.extend/
Returns:
  • The target object.
Type
object
Examples

Merge two objects

 var target = {
     foo: {
         bar: 42
     }
 }

 var object {
     foo: {
         msg: 'hello world'
     }
 }

 // Replaces object.foo with a references to target.foo.
 cog.extend(target, object);

 // The following statements are now true:
 target.foo.msg === 'hello world'
 target.foo.bar === undefined;
 target.foo === object.foo;

Merge two objects with a deep copy

 var target = {
     foo: {
         bar: 42
     }
 }

 var object {
     foo: {
         msg: 'hello world'
     }
 }

 // Merges object.foo with to target.foo, and copies properties.
 cog.extend(true, target, object);

 // The following statements are now true:
 target.foo.msg === 'hello world'
 target.foo.bar === 42;
 target.foo !== object.foo;
